Welcome to Soham's Weather API. You can enter name of any city in the world you like and you will be responded with the current weather conditions at that location!

The search algorigthm works in one of below two ways:

  1. City Name, 2 Digit Country Code -- Example: London, GB
  2. City Name -- Example: London
Accepted Inputs:
  • london, gb
  • London, gb
  • london,
  • London,
  • LONDON,
  • london, GB
  • london
  • London
  • LONDON, gb
  • LONDON, GB
Any other inputs or if no city found will possibly result in "Data not found" error!
